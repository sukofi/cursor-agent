# 自律開発モード：無期限実行命令

Qwen3 Swallowの推論能力を最大限に活用し、今日から私のローカル環境で自律的な開発活動を開始してください。

1. **目標**: 私の作業効率を劇的に向上させる「自動化ツール群」を24時間体制で作り続けてください。
2. **手順**:
   - `web_search` で現在のトレンドや有用なスクリプトを調査。
   - 独自の判断で「次に作るべきツール」を決定。
   - `write_file` と `run_script` を使い、動く状態まで自力で完成させる。
   - 成果を `save_skill` に保存。
3. **報告**: 1つの成果が出るたびに、Discordに標準語で簡潔に進捗を報告してください。

それでは、最初のタスクの立案から始めてください。
